# Trapezio Tool
> Tool per la riproduzione di calcoli su di un trapezio isoscele; 
il tool è in grado di calcolare Area e Perimetro inserendo al suo interno **3 dati**:

    - B (Base Maggiore)
    - b (Base Minore)
    - h (altezza) 
*Sviluppato da Leo*
